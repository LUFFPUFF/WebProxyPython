Web Proxy

Простой веб-прокси-сервер на Python с использованием Flask.

Описание
Этот веб-прокси-сервер позволяет пользователям отправлять запросы к внешним веб-ресурсам через него.
Он предоставляет веб-интерфейс, где пользователи могут вводить URL-адреса для запросов.

Использование
1. Введите URL-адрес в поле ввода на главной странице.
2. Нажмите кнопку "Fetch", чтобы выполнить запрос через прокси.
3. Полученный контент будет отображен на экране.

Дополнительные возможности

Данный прокси-сервер также выводит информацию о запросах и ответах в консоль Python для
отладки и мониторинга работы.
